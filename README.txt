This file explains to you how to run this script.

Before starting make sure that the Java Runtime Enviroment (JRE) is installed on your PC.

Starting of the script depends on your Operation System (OS).

Win OS:
You just need to execute the "start.bat" file.

Linux/Mac:
You need to open terminal and write following command (don't write "[]"):
java -jar [absolute path to jar file "pizzarobot-router.jar"]
and execute it.
Example: java -jar /home/user/bin/pizzarobot-router.jar

Optional: you can use this command without absolute path if you start terminal from root folder of project.
Example: java -jar pizzarobot-router.jar
